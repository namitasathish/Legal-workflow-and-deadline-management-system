## 2. Creating New Demo Logins (For Any Client)
To show how a *new* client gets access, follow these steps:

1.  **Find a Client Email**: Go to the **Clients** list as a Lawyer and copy an email (e.g., `priya.v@email.com`).
2.  **Go to Register**: Logout and go to the **Register** screen.
3.  **Select Role**: Choose **Client**.
4.  **Enter Details**:
    -   Use the copied email (`priya.v@email.com`).
    -   Create any password (e.g., `test1234`).
5.  **Submit**: The app will verify that `priya.v@email.com` exists in the Lawyer's client database and allow the registration.
6.  **Login**: Use those credentials to enter the **Client Portal** for Priya Verma.

## 3. What to Show During the Demo

### Step A: The Lawyer View (Setup)
1.  **Request a Document**: Go to a case (e.g., Arjun Sharma). Under "Document Requests", tap **+ Request** and ask for an "Income Certificate".
2.  **Add a Milestone**: Tap **+ Add** in Timeline and add "First Hearing" with a 🏛️ icon.
3.  **Send a Message**: Tap **Chat** and type "Please upload the certificate by Friday."

### Step B: The Client View (Action)
1.  **Login as Client**: Use the `demo_client@gmail.com` credentials.
2.  **Notice Dashboard**: Show the "1 Pending Request" and "Upcoming Appointments" stats.
3.  **Messaging**: Open the case, see the Lawyer's message, and reply.
4.  **Doc Upload**: Tap **Upload** on the "Income Certificate" request. Select a file from the device.
5.  **Appointments**: Go to **Appointments** → **+ Book** to request a meeting.

### Step C: Verification (Closure)
1.  **Lawyer**: Go to **Lawyer Appointments** (on Home) and **Confirm** the client's request.
2.  **Lawyer**: Go to the case and **Accept** the uploaded document.
3.  **Lawyer**: **Close** the case (choose "Won").
4.  **Client**: Go to the case and leave **Feedback** (Star rating + comment).
5.  **Lawyer**: Go to **Ratings** on Home to see the client's feedback.
